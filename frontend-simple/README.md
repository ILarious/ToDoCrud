# Frontend Simple

Простой отдельный frontend для твоего Todo API.

## Что умеет

- sign-up / sign-in
- хранение токена в `localStorage`
- CRUD для списков
- CRUD для задач внутри выбранного списка

## Быстрый запуск

Из корня проекта:

```bash
cd frontend-simple
python3 -m http.server 5173
```

Открой в браузере:

- http://localhost:5173

По умолчанию API:

- `http://localhost:8080/api/v1`

Его можно поменять в интерфейсе (поле `API`).
